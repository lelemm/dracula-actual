# Dracula for [Actual Budget](https://actualbudget.org)

> A dark theme for [Actual Budget](https://actualbudget.org).

![Screenshot](./screenshot.png)

## Install

1. In Actual, go to **Settings**.
2. Open **Appearance**.
3. Under **Custom themes**, install this GitHub repo:

```text
lelemm/dracula
```
